# Autodesk Insider Insights

Autodesk Insider Insights is a browser-based app for exploring and comparing Autodesk Water tools. It organizes product comparisons, changes and updates, and common questions that come up during evaluation and adoption, helping practitioners understand how different Autodesk Water offerings fit their workflows. It is built with React, TypeScript, Vite, and Tailwind CSS.
